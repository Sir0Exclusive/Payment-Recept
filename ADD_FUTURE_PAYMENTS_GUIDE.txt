HOW TO ADD FUTURE PAYMENTS FOR RECIPIENTS
==========================================

AUTOMATIC SETUP
===============
When a recipient registers or logs in for the first time on your website:
✓ Their section is AUTOMATICALLY created in Google Sheets
✓ Section format:

=== RECIPIENT SECTION ===  |  recipient@email.com
Receipt No | Name | Amount | Due Amount | Date | Description | Recipient Email | Payment_Status | Amount_Paid | Last Updated
(empty row ready for you to add payments)


HOW TO ADD NEW PAYMENTS (EASY METHOD)
======================================

1. Open your Google Sheet
2. Find the recipient's section (search for their email)
3. Add a new row under their section header with:

   Receipt No  | Name          | Amount | Due Amount | Date       | Description      | Recipient Email
   ----------------------------------------------------------------------------------------------------
   2007        | John Doe      | 5000   | 1000       | 2026-02-05 | Monthly service  | john@example.com

4. Leave Payment_Status and Amount_Paid empty (auto-calculated)
5. That's it!


AUTO-SYNC WORKFLOW
==================

After adding a payment in Google Sheet:

Option 1: Auto-sync is running (recommended)
   → Changes sync automatically within 30 seconds
   → Excel updates
   → Website shows new payment
   → Recipient sees it in their dashboard

Option 2: Manual sync
   → Run: sync_google_sheet.py
   → Or: double-click SYNC_AND_EXPORT.bat
   → Excel updates
   → Receipts generated
   → Website updated after git push

PAYMENT STATUS CALCULATION
==========================

Payment_Status is automatically calculated:
- Due Amount = 0  →  Status = PAID
- Due Amount > 0  →  Status = DUE

Amount_Paid is automatically calculated:
- Amount_Paid = Amount - Due Amount

Example:
  Amount: ¥5000
  Due Amount: ¥1000
  → Amount_Paid: ¥4000
  → Payment_Status: DUE


QUICK REFERENCE
===============

To add a new payment:
1. Open Google Sheet
2. Find recipient section (Ctrl+F → search email)
3. Add new row below their header
4. Fill: Receipt No, Name, Amount, Due Amount, Date, Description, Email
5. Save (auto-syncs in 30 seconds if auto-sync running)

To check all payments:
- Website → Login with your admin email → Admin Dashboard

To generate PDF receipt:
- Sync updates Excel
- Run: python generate_receipts.py
